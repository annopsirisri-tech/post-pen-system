1. ขอบเขตบริการ

Post Pen System เป็นเครื่องมือจัดการเนื้อหาบนคอมพิวเตอร์ที่อยู่ระหว่างการพัฒนา การเชื่อมบัญชีและความสามารถเผยแพร่ขึ้นอยู่กับการอนุญาตของผู้ใช้ สิทธิ API สถานะการตรวจสอบแอป และข้อกำหนดของแพลตฟอร์ม

2. ความรับผิดชอบของผู้ใช้

  • ใช้เฉพาะบัญชีที่ตนเป็นเจ้าของหรือได้รับอนุญาตอย่างถูกต้อง
  • มีสิทธิในวิดีโอ ภาพ เสียง ข้อความ เครื่องหมายการค้า และเนื้อหาทั้งหมดที่เผยแพร่
  • ปฏิบัติตามกฎหมาย ข้อกำหนดของ TikTok และกฎชุมชน
  • ตรวจสอบบัญชีปลายทาง พรีวิว วันเวลา และความเป็นส่วนตัวก่อนยืนยัน
  • ไม่ใช้ระบบเพื่อสแปม หลอกลวง ละเมิดสิทธิ หรือหลีกเลี่ยงมาตรการของแพลตฟอร์ม

3. การป้องกันเนื้อหาซ้ำ

ระบบออกแบบตามหลักหนึ่งวิดีโอต่อหนึ่งบัญชี และอาจปฏิเสธรายการที่ตรวจพบว่าเคยใช้แล้ว การตรวจดังกล่าวช่วยลดความผิดพลาด แต่ไม่ใช่การรับประกันว่าจะตรวจพบความซ้ำทุกรูปแบบ

4. บริการจากบุคคลภายนอก

การล็อกอิน การอัปโหลด และการเผยแพร่พึ่งพาบริการของ TikTok และผู้ให้บริการอื่น ความล่าช้า ข้อจำกัด การระงับบัญชี หรือการปฏิเสธเนื้อหาอาจอยู่นอกการควบคุมของ Post Pen System

5. ความพร้อมใช้งาน

ระบบอาจหยุดชั่วคราวเพื่อปรับปรุง แก้ไขข้อผิดพลาด หรือรองรับการเปลี่ยนแปลงของ API ไม่มีการรับประกันว่าบริการจะไม่มีข้อผิดพลาดหรือพร้อมใช้งานตลอดเวลา

6. การตัดการเชื่อมต่อและระงับการใช้งาน

ผู้ใช้สามารถตัดการเชื่อมต่อบัญชีได้ และเราอาจจำกัดการใช้งานที่เสี่ยงต่อความปลอดภัย ผิดกฎหมาย ละเมิดสิทธิ หรือขัดต่อข้อกำหนดของแพลตฟอร์ม

7. การจำกัดความรับผิด

เท่าที่กฎหมายอนุญาต Post Pen System ไม่รับผิดชอบต่อความเสียหายทางอ้อม การสูญเสียรายได้ การสูญเสียข้อมูล หรือผลจากการตัดสินใจของแพลตฟอร์ม ผู้ใช้ควรเก็บสำเนาไฟล์ต้นฉบับและตรวจสอบผลการเผยแพร่ด้วยตนเอง

8. ความเป็นส่วนตัว

การประมวลผลข้อมูลอยู่ภายใต้ นโยบายความเป็นส่วนตัว
