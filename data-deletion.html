ถอนสิทธิ์จาก TikTok

ผู้ใช้สามารถเข้าเมนูการตั้งค่าความปลอดภัยและแอปที่เชื่อมต่อในบัญชี TikTok แล้วถอนสิทธิ์ Post Pen System ได้ การถอนสิทธิ์อาจทำให้โทเคนเดิมไม่สามารถใช้งานได้ทันที

ขอลบข้อมูลเพิ่มเติม

ส่งคำขอผ่านหน้า Contact พร้อมระบุชื่อบัญชีที่แสดงและรายละเอียดที่เพียงพอ ต่อการค้นหาข้อมูล โดยไม่ต้องส่งรหัสผ่าน Access Token หรือ Refresh Token

ข้อมูลที่อาจถูกลบ

  • ข้อมูลบัญชีพื้นฐานที่บันทึกจากการอนุญาต
  • โทเคนและข้อมูลวันหมดอายุ
  • คิว ประวัติ และบันทึกการเผยแพร่ที่เชื่อมโยงกับบัญชี
  • ไฟล์ชั่วคราวที่อยู่ภายใต้การควบคุมของระบบ

อย่าส่งรหัสผ่าน TikTok, Client Secret, Access Token หรือ Refresh Token ผ่านช่องทางติดต่อ
